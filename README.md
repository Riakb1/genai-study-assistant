# GenAI Study Assistant

A Flask-based AI-powered study assistant deployed on AWS EC2 using the OpenAI API.

## Features
- AI-generated study answers
- Flask backend
- OpenAI API integration
- AWS EC2 cloud deployment
- GitHub version control

## Technologies Used
- Python
- Flask
- OpenAI API
- AWS EC2
- Git & GitHub
- HTML/CSS

## Deployment
The application is deployed on an Ubuntu AWS EC2 instance and accessible publicly through port 5000.
